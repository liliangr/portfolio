Cold Block Expansion

  • Category: Automation
  • Project date: 2019 - 2020
  • Client: Bavaria Colombia
  • Location: Cali, Colombia
  • Client URL: https://www.bavaria.co/del-valle

Equipment Integration

This project involved the expansion of the cold block at Cerveceria del Valle, incorporating new fermentation tanks and integrating a new centrifuge from GEA. Modifications were made to the Simatic PCS 7 and Braumat 7.0 systems.

To accomplish these objectives, I:
Modified the program for all fluid transfer sequences to include the new fermentation tanks.
Altered and created new routes using Siemens' Route Control tool to incorporate the new fermentation tanks.
Established communication with the GEA centrifuge PLC and modified sequences to integrate it into the process.
Updated existing HMI interfaces and designed new ones.
Conducted commissioning of the new instruments.
Implemented new sequences and provided support for those that were modified.
Prepared tasks reports for client oversight and as-built documentation.
